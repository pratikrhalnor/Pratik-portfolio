## Quick start

Prereqs
```bash
node >= 18
npm >= 9
```

Clone & run
```bash
git clone https://github.com/pratikrhalnor/Pratik-portfolio.git
cd Pratik-portfolio

# Frontend
cd frontend
npm install
npm run dev

# Backend (in a separate terminal)
cd ../backend
npm install
npm start
```
